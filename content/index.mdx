## What You'll Find Here

This documentation provides detailed guides, API references, and best practices for:

- **Shopify Theme Development** - Customizing and extending IMAGE SKINCARE's Shopify themes
- **Checkout Extensions** - Building and implementing checkout validation and enhancement extensions
- **React Components** - Reusable components including Recharge subscription widgets
- **Affiliate Tools** - Selector and checkout integration tools
- **Styling Guidelines** - Brand-compliant styling and component libraries

## Getting Started

<Steps>
    ### Explore the Documentation

    Browse through the different sections using the navigation menu or the cards above to find what you're looking for.

    ### Set Up Your Development Environment

    Most of our development work involves Node.js and modern web technologies. Make sure you have the following installed:
    - Node.js (v18 or later)
    - pnpm or npm
    - Git

    ### Fork and Contribute

    If you'd like to contribute to this documentation:
    ```bash
    # Fork the repository on GitHub, then clone your fork
    git clone https://github.com/Medina-Digital-LLC/IMAGE-DOCS.git
    cd IMAGE-DOCS
    ```

    ### Install Dependencies and Run Locally
    ```bash
    pnpm install
    pnpm dev
    ```
</Steps>
